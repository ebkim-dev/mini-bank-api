## JWT Deprecation

### Making Requests

Before:
```typescript
await request(app)
  .post("/accounts")
  .set("Authorization", `Bearer ${token}`)
  .send(body);
```

Now:
```typescript
await request(app)
  .post("/accounts")
  .set("x-session-id", sessionId)
  .send(body);
```

### Login (Session Generation)

Before:
```typescript
const payload: JwtPayload = {
  sub: userRecord.id,
  role: userRecord.role,
};

const token = jwt.sign(...);

await redisClient.set(`session:${sessionId}`, token, {
  EX: REDIS_SESSION_TTL_SEC,
});
```

Now:
```typescript
const payload: AuthInput = {
  actorId: userRecord.id,
  role: userRecord.role,
  customerId: userRecord.customer_id,
};

await redisClient.set(
  `session:${sessionId}`,
  encrypt(JSON.stringify(payload)),
  { expiration: { type: "EX", value: REDIS_SESSION_TTL_SEC } }
);
```

### Middleware (Session Validation)

Before:
```typescript
const token = await redisClient.get(`session:${sessionId}`);

req.user = jwt.verify(...);
```

Now:
```typescript
const raw = await redisClient.get(`session:${sessionId}`);

req.user = JSON.parse(decrypt(raw));
```

## Ownership Check

### Authorization Matrix

Before:
```
| Endpoint                 | ADMIN   | STANDARD |
| --------                 | ------- | -------- |
| POST /accounts           | ✅ | ❌ |
| GET /accounts            | ✅ | ✅ |
| GET /accounts/:id        | ✅ | ✅ |
| PUT /accounts/:id        | ✅ | ❌ |
| POST /accounts/:id/close | ✅ | ❌ |
```

Now:
```
| Endpoint                 | ADMIN   | STANDARD |
| --------                 | ------- | -------- |
| POST /accounts           | ✅ | ✅ |
| GET /accounts            | ✅ | ✅ |
| GET /accounts/:id        | ✅ (ALL Accounts) | Owned accounts only |
| PUT /accounts/:id        | ✅ (ALL Accounts) | Owned accounts only |
| POST /accounts/:id/close | ✅ (ALL Accounts) | Owned accounts only |
```

### Register

Before:
```typescript
const createdUser = await prisma.user.create(...);
```

Now:
```typescript
const customer = await prisma.customer.create(...);

const createdUser = await prisma.user.create({
  data: {
    customer_id: customer.id,
    ...
  }
});
```
